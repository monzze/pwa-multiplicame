# ng-thumb
## Angular thumbnail directive, preview uploaded images using canvas

## Installation

1. Install package with `bower i ng-thumb`
2. Add `ngThumb` to your module dependencies.

## Usage
```html
<div ng-thumb="file" height="100" width="100"></div>
```
